HTML là gì ?

HTML là ngôn ngữ đánh dấu được sử dụng để xây dựng cấu trúc nội dung của trang web. Nó sử dụng các thẻ (tags) để đánh dấu và định dạng các phần tử trên trang web, chẳng hạn như văn bản, hình ảnh, liên kết, tiêu đề, danh sách, bảng, và nhiều phần tử khác.

0.2

CSS là gì ?

CSS là một ngôn ngữ sử dụng để điều chỉnh kiểu dáng và giao diện của trang web. Nó cho phép bạn xác định màu sắc, font chữ, kích thước, khoảng cách, căn chỉnh và nhiều thuộc tính khác của các phần tử HTML.

JavaScript là gì ?

JavaScript là một ngôn ngữ lập trình thông dịch phía máy khách (client-side) được sử dụng chủ yếu trong phát triển ứng dụng web. Nó cung cấp khả năng thêm tính năng tương tác và động cho trang web.

Hệ điều hành Linux

Linux là một hệ điều hành mã nguồn mở (open source operating system) được phát triển dựa trên UNIX. Hệ điều hành Linux chạy trên nhiều loại thiết bị, bao gồm máy tính cá nhân, máy chủ, thiết bị nhúng, và nhiều hệ thống khác.
